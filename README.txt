This is a responsive Drupal theme intended to deliver both
mobile and desktop experiences without any theme switching.

http://github.com/rupl/mobile_example
mobile_example Author: Chris Ruppel

Mobile theme was built using mobile_garland: http://drupal.org/project/mobile_garland
mobile_garland Authors: Teemu Lehtinen, Mikko Hyytiainen @ Nitro FX Oy, Finland

INSTALLATION

1. Download the latest stable mobile_garland-release for your Drupal branch.
2. Extract the contents to your sites/all/themes -folder.
3. Enable the theme in
     Administer > Site building > Themes

MOBILE THEMES

Information about creating mobile themes building on this simple theme.
  http://drupal.org/project/mobile_garland
